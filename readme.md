# Borsa Veri Analizi

Python ile BIST hisselerinin (Tupraş, THY) tarihsel verilerini çekip 
analiz eden öğrenme amaçlı bir proje.

## Özellikler
- Yahoo Finance üzerinden gerçek zamanlı hisse verisi çekme
- Temel istatistiksel özet (ortalama, min, max)
- 5 ve 10 günlük hareketli ortalama grafiği
- İki hisseyi normalize ederek karşılaştırma
- Korelasyon analizi

## Kullanılan Teknolojiler
- Python
- pandas
- yfinance
- matplotlib

## Çalıştırma
\```
pip install -r requirements.txt
python borsa_analiz.py
\```
